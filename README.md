# SupleHubZN
E-commerce MERN stack para SupleHubZN. Incluye catálogo dinámico de suplementos con filtros, carrito persistente, autenticación JWT y pasarela de pago. Panel de administración para gestión de stock y órdenes, diseñado con Tailwind CSS para una experiencia responsiva y profesional orientada al sector fitness. 🚀🏋️‍♂️
